This is a reference for C++ standard data structures and their methods.  
Currently four data structures are covered:
* Hash maps
* Queues
* Sets
* Stacks

Feel free to add to the list. There are still many that can be added such as vectors, arrays, pairs and tupples. 

## Contribution instructions
1. Fork and clone the repository
2. Add your scripts
3. Commit and push
4. Create a pull request
6. Wait for pull request to merge
